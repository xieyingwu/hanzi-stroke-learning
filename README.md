# 🐼 汉字笔顺学堂

一款专为小学生设计的汉字笔顺学习网页应用，支持动画演示、逐笔练习和分类学习，让写字更有趣！

## ✨ 功能特色

- **📖 丰富的汉字库**：内置 400+ 常用汉字，涵盖拼音与词义注释
- **🎬 笔顺动画**：基于 [Hanzi Writer](https://hanziwriter.org/) 提供精准的笔顺动画演示
- **✏️ 田字格展示**：带田字格参考线，帮助练习标准写字姿势
- **🔢 逐笔练习**：支持逐笔播放，随时暂停、重放，掌握每一笔的走向
- **📚 分类学习**：按年级或主题分类，循序渐进
- **⭐ 星星激励**：学会一个字获得星星奖励，激发学习动力
- **📊 学习进度**：实时展示已学字数、今日进度和连续打卡天数
- **💾 本地记录**：学习记录保存在浏览器本地，随时继续

## 🚀 快速开始

本项目为纯前端静态页面，无需安装任何依赖，直接用浏览器打开即可。

```bash
# 克隆仓库
git clone https://github.com/xieyingwu/hanzi-stroke-learning.git

# 用浏览器打开
open index.html
```

或直接通过 GitHub Pages 在线访问（如已部署）。

## 🛠️ 技术栈

- **原生 HTML / CSS / JavaScript**：零框架依赖
- **[Hanzi Writer v3.5](https://hanziwriter.org/)**：提供汉字笔顺数据与动画渲染
- **hanzi-writer-data v2.0**：通过 jsDelivr CDN 按需加载字形数据
- **LocalStorage**：持久化存储学习记录与星星数

## 📱 界面说明

1. **顶部导航栏**：显示应用名称和当前星星总数
2. **统计卡片**：已学字数 / 总字数 / 连续打卡天数
3. **进度条**：当前分类的今日学习进度百分比
4. **分类选择器**：横向滚动切换汉字分类
5. **汉字网格**：4 列卡片展示，已学字有 ✅ 标记
6. **学习弹层**：点击汉字卡片弹出，包含：
   - 汉字大图 + 拼音 + 笔画数
   - 田字格 + 笔顺动画画布
   - 笔画进度指示点
   - 播放 / 逐笔 / 重来 控制按钮
   - "我记住了！"确认按钮（获得星星）

## 📜 数据来源

汉字字形与笔顺数据来源于 [Hanzi Writer](https://hanziwriter.org/)（基于 [makemeahanzi](https://github.com/skishore/makemeahanzi)），拼音与词义由本项目自定义。
