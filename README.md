# 🐼 汉字笔顺学堂

一款专为儿童设计的汉字笔顺学习网页应用，支持在线逐笔动画演示、自主练习、学习进度追踪，无需安装，打开即用。

---

## 📂 文件说明

| 文件 | 汉字数量 | 说明 |
|------|----------|------|
| `hanzi-stroke-learning.html` | **3500 字** | 完整增强版，覆盖常用汉字 3500 字，16 大分类 |
| `index.html` | **403 字** | 基础版，按年级划分（一至四年级）+ 主题分类 |

---

## ✨ 功能特性

- **笔顺动画演示**：基于 [Hanzi Writer](https://hanziwriter.org/) 逐笔高亮演示，清晰直观
- **分类浏览**：支持多维度分类筛选，快速找到目标汉字
- **学习进度保存**：利用 `localStorage` 记录已学汉字，刷新页面不丢失
- **星星奖励系统**：学会每个汉字获得星星，激励持续学习
- **进度统计**：实时展示当前分类学习完成百分比
- **底部导航**：✏️ 学习 / 🔁 复习 / 🏆 排行 / 🐼 我的
- **移动端优先**：针对手机屏幕深度适配，支持触屏操作
- **可爱 UI**：云朵背景、彩色卡片、吉祥物小胖🐼互动提示

## 🚀 使用方法

1. 下载仓库中的 HTML 文件（推荐使用 `hanzi-stroke-learning.html`）
2. 用浏览器直接打开，或部署到任意静态托管平台（GitHub Pages、Vercel 等）
3. 点击汉字卡片，即可观看笔顺动画并标记为已学

> **提示**：也可直接通过 GitHub Pages 访问，无需下载。
