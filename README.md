# 🐼 汉字笔顺学堂

一款专为儿童设计的汉字笔顺学习网页应用，支持逐笔动画演示、分类练习、学习进度追踪。无需安装，用浏览器打开即可使用。

---

## 📂 文件说明

| 文件 | 汉字数量 | 分类数 | 适合人群 |
|------|----------|--------|----------|
| `hanzi-stroke-learning.html` | **3500 字** | 16 类 | 全阶段学习者，覆盖常用汉字 |
| `index.html` | **403 字** | 16 类 | 小学阶段（一至四年级）+ 主题分类 |

---

## ✨ 功能特性

两个版本均包含以下完整功能：

| 功能 | 说明 |
|------|------|
| 🖊️ **笔顺动画** | 基于 [Hanzi Writer 3.5](https://hanziwriter.org/) 逐笔高亮演示，支持「播放」「下一笔」「重来」控制 |
| 📐 **田字格显示** | 动画区内叠加 SVG 田字格，清晰呈现笔画位置和结构 |
| 📊 **拼音 + 释义 + 笔画数** | 弹窗展示汉字拼音、中文释义、笔画总数 |
| 🌟 **星星奖励** | 每学会一个汉字获得 3 颗星，顶部实时显示累计星星数 |
| ✨ **星星爆炸特效** | 标记学会时触发全屏星星飞散动画，成就感满满 |
| 📅 **连续学习天数** | 统计栏显示累计连续打卡天数，培养学习习惯 |
| 📈 **今日进度条** | 实时显示当前分类已学/总字数及完成百分比 |
| 📂 **16 大分类筛选** | 按年级或主题切换，聚焦不同场景词汇 |
| 💾 **本地进度保存** | 使用 `localStorage` 持久化已学汉字、星星数、连续天数，刷新不丢失 |
| 📱 **移动端优先** | `viewport` 限制缩放，触屏操作流畅，适配各尺寸手机屏幕 |
| 🐼 **吉祥物互动** | 小胖🐼 根据操作动态显示鼓励提示语 |
| 🔔 **Toast 消息** | 操作反馈轻提示，无干扰用户体验 |
| 🧭 **底部四标签导航** | ✏️ 学习 / 🔁 复习 / 🏆 排行 / 🐼 我的（复习/排行/我的功能即将推出） |

## 🚀 使用方法

1. 下载仓库中的 HTML 文件（汉字量更多推荐用 `hanzi-stroke-learning.html`，小学入门推荐用 `index.html`）
2. 用浏览器（Chrome / Safari / Edge）直接打开，或部署到 GitHub Pages、Vercel 等静态托管平台
3. 点击汉字卡片 → 观看笔顺动画 → 点击「🌟 我记住了！」标记学会并获得星星奖励

> **提示**：学习进度、星星数、连续天数均自动保存在浏览器本地，清除缓存会重置记录。

## 🛠️ 技术栈

| 技术 | 用途 |
|------|------|
| [Hanzi Writer 3.5](https://hanziwriter.org/) | 汉字笔顺数据库与逐笔动画渲染 |
| 原生 HTML / CSS / JavaScript | 无框架依赖，单文件即可运行 |
| SVG | 田字格叠加层绘制 |
| `localStorage` | 学习进度、星星、连续天数本地持久化 |

## 📸 界面亮点

- **顶部导航**：渐变橙粉色背景 + 实时星星计数
- **统计卡片**：已学会 / 总字数 / 连续天数三栏并排展示
- **今日进度条**：动态渐变填充，直观显示完成率
- **汉字卡片网格**：已学字高亮区分，一目了然
- **弹出学习面板**：田字格 + 笔顺动画 + 拼音/释义/笔画数 + 控制按钮
- **星星爆炸特效**：学会汉字后触发全屏 ⭐🌟✨ 飞散庆祝
- **吉祥物小胖🐼**：实时互动气泡提示，陪伴学习全程
- **云朵渐变背景**：天蓝色渐变 + 白色云朵光晕，清新可爱
